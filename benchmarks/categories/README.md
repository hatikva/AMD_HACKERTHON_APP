This Version 2 suite is the repository-owned offline qualification benchmark for Version 5 primary task-category model qualification. It contains eight canonical categories, five progressively difficult tasks per category, and 40 tasks total.

The benchmark is not live input. Do not copy it to `/input/tasks.json`, `/output`, Docker live-task paths, or model-visible runtime context. Evaluation metadata is withheld from tested models and used only by the qualification runner.

Version 1 benchmark files are retired and are not active qualification inputs.
